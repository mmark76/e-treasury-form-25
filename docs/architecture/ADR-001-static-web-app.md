# ADR-001: Στατική web εφαρμογή για το αρχικό MVP

- **Κατάσταση:** Εγκεκριμένη για το MVP
- **Ημερομηνία:** 2026-06-25

## Πλαίσιο

Το πρώτο λειτουργικό στάδιο χρειάζεται χειροκίνητη καταχώριση, τοπικά πρότυπα ανά πεδίο, υπολογισμούς και εκτύπωση. Δεν υπάρχει ακόμη απαίτηση για κοινή βάση δεδομένων ή πολλαπλούς χρήστες.

## Απόφαση

Το MVP υλοποιείται ως στατική web εφαρμογή με HTML, CSS και native JavaScript modules.

Τα πρότυπα και το προσωρινό έντυπο αποθηκεύονται στο `localStorage`. Οι υπολογισμοί, η απόδοση ποσού ολογράφως, η διαχείριση προτύπων και η προεπισκόπηση παραμένουν σε ξεχωριστά modules.

## Συνέπειες

### Θετικές

- ελάχιστη πολυπλοκότητα και μηδενικό κόστος backend,
- εύκολη φιλοξενία σε GitHub Pages ή οποιονδήποτε στατικό server,
- μικρά και ανεξάρτητα αρχεία,
- εύκολη μελλοντική αντικατάσταση του local storage από API.

### Περιορισμοί

- τα δεδομένα δεν συγχρονίζονται μεταξύ browsers ή υπολογιστών,
- η εκκαθάριση δεδομένων του browser αφαιρεί τα αποθηκευμένα πρότυπα,
- δεν υπάρχει κεντρικός έλεγχος αρίθμησης ή πρόσβασης.

## Μελλοντική εξέλιξη

Όταν προκύψει πραγματική ανάγκη συγχρονισμού, θα προστεθεί ξεχωριστό infrastructure module και API χωρίς να μεταφερθεί επιχειρησιακή λογική μέσα στο UI.
